# Personal Presentation — siuldevelop

A minimal personal presentation page built with vanilla HTML, CSS, and JavaScript.

## Features

- Smooth inertial scroll
- Dynamic navbar with frosted-glass effect on scroll
- Typewriter animation (pauses on hover)
- Scroll-triggered skill cards
- Contact form via Formspree
- Back to top button

## Stack

HTML · CSS · JavaScript
